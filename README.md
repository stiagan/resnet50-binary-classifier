# ResNet50 Binary Image Classifier

A deep learning image classifier built with PyTorch and ResNet50.

## Features
- Transfer learning with ResNet50
- Cat vs Dog classification
- Gradio web interface

## Installation

pip install -r requirements.txt

## Kaggle Notebook
https://www.kaggle.com/code/ummadd/cat-dog-classifier

## Download the Model
https://www.kaggle.com/models/ummadd/cat-dog-classifier

## Download the Dataset
https://www.kaggle.com/datasets/karakaggle/kaggle-cat-vs-dog-dataset

## Run the app

python app.py
